
# my-app

## Project Description
This is a project built with a backend and a frontend.

## Quick Start Guide
1. Clone this repository
2. `docker-compose up` to start the project
3. Access the frontend at `http://localhost:3000`

## Manual Setup Instructions
1. Install Node.js and npm
2. Install PostgreSQL
3. Create a new database
4. Run `npm run build` and `npm run start` in the frontend directory
5. Run `npm run start` in the backend directory
